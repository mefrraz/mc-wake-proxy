DuckDNS — Dynamic DNS for Minecraft

DuckDNS is a free dynamic DNS service. It gives you a permanent domain name (like mefrraz.duckdns.org) that always points to your home IP — even when your ISP changes it.

Setup

  1. Register at duckdns.org (sign in with GitHub/Google)
  2. Create a subdomain (e.g. minecraftamigos)
  3. Install the update script on your Pi so DuckDNS always knows your IP (see full guide)
  4. Forward TCP port 25565 on your router to the Pi's local IP
  5. That's it — all subdomains (*.minecraftamigos.duckdns.org) resolve to your network automatically

Multi-Server

Use different hostnames for different Minecraft servers — the proxy routes by hostname. Example:

minecraftamigos.duckdns.org→ Main server (port 25565)
survival.minecraftamigos.duckdns.org→ Survival server (port 25566)
creative.minecraftamigos.duckdns.org→ Creative server (port 25567)

Proxmox Setup

Create an API token with minimum permissions for LXC management.

  1. Log into Proxmox: https://PROXMOX_HOST:8006
  2. Datacenter → Permissions → API Tokens → Add
  3. User: root@pam — Token ID: mc-wake-proxy
  4. Copy the secret UUID immediately
  5. Permissions → Add → API Token Permission → Path /vms/LXC_ID → Role PVEVMUser
  6. Run hostname on Proxmox — use that for PROXMOX_NODE

Full guide: docs/proxmox-setup.md

Crafty Controller Setup

Generate an API token and enable status visibility.

  1. Open Crafty: https://CRAFTY_HOST:8443
  2. Username menu → API Tokens → Create
  3. Copy the token immediately
  4. For each server: Server Configuration → Show Status → ON
  5. Server ID is in the URL: .../server_detail?id=UUID

Full guide: docs/crafty-setup.md
